# Сравнение алгоритмов растеризации отрезков

| Критерий | Пошаговый алгоритм | Алгоритм ЦДА | Алгоритм Брезенхема |
|----------|-------------------|-------------|-------------------|
| **Математическая основа** | Линейная функция с округлением | Дифференциальное уравнение Δy/Δx | Целочисленная ошибка между идеальной линией и пикселями |
| **Тип связности** | 8-связная | 8-связная | 8-связная |
| **Используемая арифметика** | Вещественные числа (умножение, округление) | Вещественные числа (деление на каждом шаге) | Только целочисленные операции |
| **Операции на шаг** | Умножение + округление | Деление + сложение | 2-3 целочисленные операции |
| **Качество линии** | Среднее (из-за погрешностей округления) | Среднее (зависит от реализации) | Высокое (наиболее точный) |
| **Быстродействие** | Низкое | Среднее | Высокое |
| **Требования к памяти** | 2 вещественные переменные | 2 вещественные переменные | 3-4 целочисленные переменные |
| **Наличие деления** | Нет | Да (на каждом шаге) | Нет |
| **Наличие округления** | Да (на каждом шаге) | Зависит от реализации | Нет |
| **Поддержка всех направлений** | Требует доработки | Существуют версии для всех октантов | Сводится к первому октанту через симметрию |
| **Практическое применение** | Учебные примеры | Практически не используется | Стандарт в графических системах |
| **Особенности** | Наиболее прост для понимания | Основан на численных методах | Имеет модификации для других фигур |
